üöÄ Getting Started

Welcome to the production calendar system! This tool allows crew members to easily access the latest shooting schedules without needing to create accounts.

Two Ways to Access:

  1. Access Code: Enter an 8-character code like HAMLET24
  2. Direct Link: Click a link shared by the production office

üîë Using Access Codes

How to Get an Access Code:

  • Access codes are provided by the production office
  • Check your email for schedule updates
  • Look for messages from the 1st AD or production coordinator
  • Ask a department head if you haven't received one

Entering Your Code:

  1. Visit the access page
  2. Enter your 8-character code (like HAMLET24)
  3. Codes are not case-sensitive
  4. Click "Access Calendar" to view the schedule
Tip: Bookmark the calendar page after accessing it for quick future access!

üîó Using Direct Links

Production may share direct links via email or text message. These links take you straight to the calendar without entering a code.

Link Format:

https://yourapp.com/calendar/abc123def456
  • Links work immediately - no code entry needed
  • Bookmark the link for easy access
  • Share with other crew members if authorized

üìÖ Using the Calendar

Navigation Features:

  • Today Button: Jump to today's date
  • Print Button: Print the schedule
  • Filters: Show/hide specific information

Calendar Information:

  • Shoot Days: Days with principal photography
  • Prep Days: Setup and preparation days
  • Wrap Days: Breakdown and wrap activities
  • Call Times: When to arrive on set
  • Locations: Where filming takes place
  • Departments: Which departments are needed
Note: This is a read-only calendar. You cannot make changes to the schedule.

üîß Troubleshooting

Access Code Issues:

  • "Invalid access code": Check spelling and try again
  • Code not working: Contact the production office for a new code
  • Expired code: Codes may expire - request a current one

Link Issues:

  • Broken link: The schedule may have been updated with a new link
  • Page not found: Contact production for the current link

General Issues:

  • Mobile display: The calendar is optimized for mobile viewing
  • Printing: Use landscape orientation for best results
  • Updates: Refresh the page to see the latest schedule changes
Still having trouble? Contact the production office - they're there to help!

üìû Getting Help

If you're having trouble accessing the calendar or have questions about the schedule:

  • Production Office: Your first point of contact
  • 1st Assistant Director: For schedule-related questions
  • Production Coordinator: For access issues
  • Department Heads: For department-specific information

When contacting for help, please mention:

  • The production name
  • Your department
  • The specific issue you're experiencing
  • Whether you have an access code or link
